# Data Directory

This directory is intentionally left empty in the repository.

Due to dataset size and licensing constraints, the processed BigVul dataset is **not distributed here**.

Please download the dataset from the official MoEVD artifact:

https://zenodo.org/records/11661787

After downloading, place the following files here:

- train.parquet  
- val.parquet  
- test.parquet  

Refer to the main README for detailed instructions.
